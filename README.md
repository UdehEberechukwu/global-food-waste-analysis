## Project Overview

This project analyzes global food waste patterns using data from the UNEP Food Waste Index.

The dataset was imported into SQL Server, where data cleaning and transformation were performed. Text fields were standardized, data types were converted, confidence levels were reviewed, and SQL views were created to support analysis and reporting.

To improve reliability, the analysis focuses on countries classified as High Confidence and Medium Confidence.

Power BI was used to build an interactive dashboard that explores food waste patterns across countries, regions, and sectors.

## Objectives

* Identify countries with the highest and lowest food waste per capita
* Compare food waste levels across regions
* Analyze household, retail, and food service contributions
* Develop KPIs and interactive visualizations
* Generate actionable insights from global food waste data

## SQL Tasks Performed

* Created analytical SQL views for reporting and dashboard development

## Dashboard Features

* Top 10 Countries by Food Waste per Capita
* Lowest Food Waste Countries
* Average Food Waste by Region
* Total Waste by Region
* Household Contribution by Country
* Country Sector Breakdown
* Global Food Waste Composition
* Interactive Country and Region Filters
* KPI Cards for Average Waste and Countries Analyzed

## Key Insights

* Malaysia recorded the highest food waste per capita among analyzed countries.
* Household waste represents the largest share of global food waste.
* Food waste levels vary significantly across countries and regions.
* A relatively small group of countries accounts for disproportionately high food waste levels.
